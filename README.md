
# Japan Trip Constructor

Офлайн-конструктор путешествий на чистом HTML / CSS / JavaScript.

## Что внутри

- `index.html` — главная страница и быстрый старт
- `planner.html` — конструктор поездок
- `osaka.html`, `fuji.html`, `tokyo.html`, `shanghai.html` — демо-страницы городов
- `budget.html`, `visa.html`, `glossary.html`, `contacts.html`, `toilet-map.html` — полезные утилиты
- `CSS/style.css` — общая тема и адаптивная вёрстка
- `js/data.js` — демо-данные
- `js/currency.js` — курсы и форматирование
- `js/app.js` — вся логика приложения
- `manifest.json`, `sw.js` — PWA-обвязка

## Как открыть

Открывай через обычный локальный сервер или через хостинг со статическими файлами. При первом открытии приложение кэширует основные ресурсы и дальше работает офлайн.
