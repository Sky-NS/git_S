🗣️ Важные фразы для экстренных ситуаций

🇯🇵 Японский:

• Помогите! — 助けて！ (Тасукэтэ!)

• Вызовите полицию / скорую. — 警察 / 救急車を呼んでください。 (Кэйсацу / Кюкю:ся ёндэ кудасай.)

• Мне нужен врач, говорящий по-русски. — ロシア語のできる医者が必要です。 (Росиа-го но дэкиру ися га хицуё: дэс.)

• Где ближайшая больница? — 一番近い病院はどこですか？ (Итибан тикай бё:ин ва доко дэсука?)

🇨🇳 Китайский (мандарин):

• Помогите! — 救命！ (Jiùmìng!)

• Вызовите полицию / скорую. — 请叫警察 / 救护车。 (Qǐng jiào jǐngchá / jiùhùchē.)

• Мне нужен русскоговорящий врач. — 我需要会说俄语的医生。 (Wǒ xūyào huì shuō éyǔ de yīshēng.)

• Где ближайшая больница? — 最近的医院在哪里？ (Zuìjìn de yīyuàn zài nǎlǐ?)

💡 Полезные приложения:
• VoiceTra — бесплатный голосовой переводчик (японский/английский/русский)
• Google Translate — с офлайн-пакетами для японского и китайского
• Maps.me — офлайн-карты для навигации без интернета

📌 На заметку:
• Номера экстренных служб работают без SIM-карты, если есть сигнал любой сети.
• При звонке в полицию или скорую лучше попросить англоговорящего оператора: "English, please".
• В Японии и Китае аптеки (薬局 / 药店) могут отпускать лекарства только по рецепту. Не забудьте взять с собой аптечку первой необходимости.
